# SB Trader Client

React client for `SB Trader` project.

## Getting Started

Install dependencies
```
pnpm install
```

Run client
```
npm run dev
```

## Backend and Blockchain integration

1. Backend url and port: `http://127.0.0.1:3000`
2. Uses Sui package deployed to `testnet`
